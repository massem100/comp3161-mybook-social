# comp3161-mybook-social
	- place the mybook.sql (found in the app folder) in your local disk 
	- create your database mybook and run mybook.sql to create the tables. 
	- open cmd and run the following code: 
		1.) python  -m venv venv 
		2.) .\venv\scripts\activate 
		3.)pip install  -r requirements.txt 
		4.)pip freeze > requirements.txt 
		5.)Python run.py
	- open the init.py file and verify that the Mysql user name and mysql password matches your database 
	-open web browser and type http://localhost:8080/ 
	- sign up and login
